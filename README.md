
# Form Validation Project

A responsive Form Validation project built using HTML, CSS, and JavaScript.  
This project validates user inputs before submitting the form.

---

## 🚀 Features

- Username validation
- Email validation
- Password validation
- Confirm password checking
- Error message display
- Responsive design
- Clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

```bash
form-validation/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

---

## ⚙️ Validation Rules

The form checks:

- Empty fields
- Valid email format
- Minimum password length
- Matching passwords

Error messages appear if the input is invalid.

---

## ▶️ How to Run

1. Download or clone the repository

```bash
git clone https://github.com/your-username/form-validation.git
```

2. Open the project folder

3. Run the `index.html` file in your browser
